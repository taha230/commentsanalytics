What is CommentsAnalytics Chrome Extension?

CommentsAnalytics chrome extension tool can be a useful way to extract comments from platforms such as Youtube. This tool can be installed as an extension to the Chrome browser, and once installed, it allows users to easily go to video page and export comments from a Youtube video with a simple click of a button.

These comments can make it easier for businesses to extract and analyze customer feedback from their Youtube videos in order to gain insights and improve their content, marketing strategy and overall customer experience.

Furthermore, this can also help businesses to track their brand reputation by monitoring the comments on their videos and general perception of their brand.

Get Comments in 30 seconds

Install CommentsAnalytics chrome extension to extract comments of Youtube video page and store the JSON file in your storage. You can use this JSON file in your CommentsAnalytics dashboard to create a new Bulk analysis request.

1. Install

Download it from Chrome webstore and install it on your browser. Please be sure to have latest chrome update.(> Version 88.0.4324.182)

2. Go to your comments page

You need to go to your comments URL. For example you want to download all comments of your Youtube video. Get the URL of your video page.

3. Scroll down the page until all the comments have been loaded

You need to "scroll down" the comments page until all the comments you want to export are loaded. Sometimes, webpages only load a certain number of comments at a time, so you'll need to keep scrolling to ensure that all the comments have been loaded.

4. Export the comments

After the scroll down goes to the end of the page, all comment has extracted. By right click on the window and click on the "Export Comments" context menu, you can get export all comments in a JSON file.

You can use this JSON file in creating a new "Bulk request" by uploading it on your CommentsAnalytics dashboard.
