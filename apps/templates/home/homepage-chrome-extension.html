What is CommentsAnalytics Chrome Extension?

CommentsAnalytics chrome extension tool can be a useful way to extract comments from platforms such as YouTube videos. This tool can be installed as an extension to the Chrome browser, and once installed, it allows users to easily select video URL and export comments from a YouTube video with a simple click of a button.

These comments can make it easier for businesses to extract and analyze customer feedback from their YouTube videos in order to gain insights and improve their content, marketing strategy and overall customer experience.

Furthermore, this can also help businesses to track their brand reputation by monitoring the comments on their videos and general perception of their brand.

Install CommentsAnalytics chrome extension to extract comments of Youtube video page and store the JSON file in your storage. You can use this JSON file in your CommentsAnalytics dashboard to create a new Bulk analysis request.

3. Paste URL on extension textbox

Click on the CommentAnalytics installed extension and Paste your URL on the textbox. Click on the "Export Comments" button to start scrapping the comment in the new window.

4. Export the comments

The scrapping module starts to scroll down the page in the new window. Do not close the window untill the module ends to the last comment.

After the scroll down module goes to the end of the page, all comment has extracted. By right click on the window and click on the "Export Comments" menu, you can get export all comments in a json file.

You can use this json file in creating a new "Bulk" request by uploading it on your CommentsAnalytics dashboard.
